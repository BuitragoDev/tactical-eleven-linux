# ⚽ Tactical Eleven

> Simulador de gestión futbolística estilo *Football Manager* desarrollado en **Unity 6** con **UI Toolkit** y base de datos **SQLite**.

---

## 🧠 Descripción

**Tactical Eleven** es un juego de gestión deportiva donde tomas el control total de un club de fútbol: desde la planificación táctica hasta las decisiones financieras y de mercado.

El proyecto está diseñado con una arquitectura modular inspirada en patrones tipo **WPF**, adaptada a Unity mediante **UXML + C#**, permitiendo una navegación fluida entre pantallas y una UI escalable.

---

## 🚀 Características principales

* 📊 Sistema de clasificación y resultados
* 💰 Gestión financiera del club
* 🔄 Sistema de fichajes y negociaciones
* 🧩 Interfaz modular con múltiples pantallas
* 🗂️ Base de datos persistente con SQLite
* 🧱 Arquitectura UI basada en UI Toolkit

---

## 🛠️ Tecnologías utilizadas

* **Unity 6**
* **C#**
* **UI Toolkit (UXML + USS)**
* **SQLite**
* **Visual Studio / VS Code**

---

## 🧱 Estructura del proyecto

```bash
📁 Assets/              # Recursos del juego (scripts, UI, etc.)
📁 Packages/            # Dependencias de Unity
📁 ProjectSettings/     # Configuración del proyecto
📁 UIElementsSchema/    # Definiciones UI Toolkit
📄 TacticalEleven.sln   # Solución principal
```

---

## 🧩 Arquitectura UI

El sistema de interfaz sigue un enfoque desacoplado:

* Cada pantalla está definida en **UXML**
* Controladores en **C#** gestionan la lógica
* Navegación dinámica entre vistas
* Uso de contenedores anidados para layouts complejos

Inspiración: patrones tipo **MVVM adaptados a Unity**
